Display Fibonacci Series. (Tiếng việt: hiển thị dãy Fibonacci của n số đầu tiên (do người dùng nhập vào.) => Nguồn: https://www.programiz.com.

Suggestions

+ Sử dụng vòng lặp for. + Dãy Fibonacci là một dãy trong đó số hạng tiếp theo là tổng của hai số hạng trước đó. Hai số hạng đầu tiên của dãy Fibonacci là 0 theo sau là 1.

Nhắc lại kiến thức

Vòng lặp for 
                        
for(gia_tri_dau; bieu_thuc; bien_tang/giam) {
    khối code
}

Code mẫu: 

                        
#include < stdio.h >
int main() {
    int n;
    printf("so phan tu: ");
    scanf("%d", &n);
    int st = 0, nd = 1;
    printf("dãy Fibonacci: ");
    printf ("%d %d ", st, nd);
    int next = st + nd;
    for (int i = 3; i <= n; i++) {
        printf("%d ", next);
        st = nd;
        nd = next;
        next = st + nd;
    }
    return 0;
}
